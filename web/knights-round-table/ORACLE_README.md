# Excalibur Oracle Web Interface

Web portal interface for accessing the Excalibur Protocol Oracle.

## Overview

The Oracle web interface provides an intuitive way for users to:
- Ask questions about the protocol
- Receive intelligent guidance on mining, forging, vaults, and treasury
- Get oracle divinations and wisdom
- View protocol status and statistics

## Files

- **oracle.html** - Oracle interface HTML page
- **oracle.js** - Oracle client-side JavaScript logic
- **index.html** - Updated Knights' Round Table with oracle link

## Features

### 1. Oracle Query System
- Free-form question input
- Quick query buttons for common questions
- Intelligent response based on query content
- Category-based knowledge retrieval

### 2. Oracle Divination
- Receive wisdom from the oracle
- Protocol status updates
- Arthurian-themed messages

### 3. Protocol Statistics
- Real-time oracle status
- Taproot address display
- Query count tracking
- Protocol version info

## Query Topics

The oracle can answer questions about:

- **Mining**: How to mine, algorithm details, requirements
- **Forging**: Forge process, rewards, fees
- **Vaults**: Taproot addresses, generation, security
- **Treasury**: Admin credentials, security, access
- **Axiom**: Protocol axiom, usage, importance
- **Difficulty**: Target requirements, algorithm
- **Taproot**: Address verification, integration

## Usage

### Access the Oracle

1. Navigate to `oracle.html` from Knights' Round Table
2. Or directly access: `/web/knights-round-table/oracle.html`

### Ask a Question

1. Type your question in the text area
2. Click "Consult the Oracle" or press Enter
3. View the oracle's wisdom and detailed response
